This project is a collection of Kafka Streams applications that demonstrate various use cases and features of Kafka Streams. Each application is designed to process and analyze streaming data in real-time, showcasing the power and flexibility of Kafka Streams for building scalable and resilient stream processing applications.

This project uses Docker Compose to manage the application and its dependencies.
Before launching any application, please make sure to launch the necessary dependencies using _docker compose up_ command in the root directory of this project. This will start the Kafka cluster, Schema Registry, and other necessary services.
Then the application can be launched by running the main classes of the applications from your IDE.
